# claude-skills
Personal compilation of Claude skills

Claude skills本质上是一个文件夹，它将执行步骤、工具使用方式以及必要的知识材料整合在一起，形成一个结构化的“技能包”，能让 AI 每次都能稳定、专业、不出错地完成固定工作。它的结构有四大部分：skill.md是核心文件，定义了skill的描述、功能和执行逻辑；references是参考文档，存放供claude查阅的“知识”，如API文档、代码规范等；examples是模板文件，用于生成固定格式的输出；scripts是可执行脚本文件夹。
